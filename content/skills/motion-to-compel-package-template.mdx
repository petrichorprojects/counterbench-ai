## Steps

1. Create a deficiency table: Request | Response | Defect | Relief.
2. Summarize meet-and-confer (dates + outcomes).
3. Draft concise relief request and proposed order outline.

## Copy/Paste Prompt

```text
Build a motion-to-compel package from the following materials.

Inputs:
- Jurisdiction: [ ]
- Discovery requests (paste): [ ]
- Responses/objections (paste): [ ]
- Meet-and-confer notes (paste): [ ]

Output:
1) Deficiencies table (Request | Response | Defect | Relief requested)
2) Meet-and-confer chronology
3) Draft outline for motion + proposed order

Do not invent rule citations. Mark them [VERIFY LOCAL RULE].
```
