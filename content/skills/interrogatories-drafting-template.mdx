## Steps

1. Identify disputed issues (liability, causation, notice, comparative negligence).
2. Draft interrogatories that force names, dates, and document IDs.
3. Add an authentication/ESI section (where appropriate).

## Copy/Paste Prompt

```text
Create interrogatory starters for a plaintiff-side PI case.

Inputs:
- Jurisdiction: [ ]
- Case type: [AUTO / PREMISES / PRODUCT / ETC]
- Defendants: [LIST]
- Known defenses: [PASTE]

Output:
- 20–30 interrogatory starters grouped by theme
- Each question should be short, fact-focused, and designed to generate follow-up RFPs
- Include an "insurance/coverage" section
```
