## Checklist

Liability
- [ ] Police report / incident report
- [ ] Photos/video
- [ ] Witness statements
- [ ] Diagram / scene

Damages
- [ ] Treatment summary
- [ ] Key records (ER, imaging, specialists)
- [ ] Bills and totals
- [ ] Wage loss documents
- [ ] Future care/permanency support

Packaging
- [ ] Demand letter
- [ ] Exhibit index
- [ ] Single PDF, bookmarked

## Quality Checks
- Totals match in letter and exhibits.
- Index makes it skimmable in 60 seconds.
- No missing imaging reports.
