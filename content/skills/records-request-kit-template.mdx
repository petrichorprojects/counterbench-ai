## Steps

1. List record sources by category (medical, employment, incident, insurance).
2. For each source: what to request + why it matters.
3. Add deadline risk items (video overwrite, retention).

## Output Template

```text
Records Request Kit

Incident
- Police report
- EMS run sheet
- Photos/video

Medical
- ER
- Imaging
- PT
- Ortho

Employment
- Wage + attendance
- Job description

For each: Request | Contact | Authorization needed? | Proves | Status
```
