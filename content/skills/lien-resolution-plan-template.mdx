## Steps

1. Identify lien sources (private, Medicare/Medicaid, ERISA, workers comp).
2. Request itemized lien statements early.
3. Track reductions/waivers and required documentation.
4. Build a closing statement that matches the final lien numbers.

## Output Template

```text
Lien Tracker

- Provider/Plan | Contact | Amount Claimed | Basis | Docs Needed | Status | Next Action | Due Date

Close-out checklist
- [ ] Confirm all lienholders identified
- [ ] Request final statements
- [ ] Negotiate reductions
- [ ] Confirm payoff instructions
- [ ] Update settlement statement
```

## Quality Checks
- Every lien has an owner/contact + next action.
- Settlement statement uses final numbers, not estimates.
