## Steps

1. Restate the claim exactly.
2. Identify required jurisdiction and court level.
3. For each authority:
   - Confirm name and citation.
   - Confirm key proposition in context.
   - Note any limitations, dissent, or overruling history.
4. Return a short verdict: accurate / partially accurate / unsupported.
