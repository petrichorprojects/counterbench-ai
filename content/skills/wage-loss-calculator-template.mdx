## Steps

1. Determine hourly vs salary vs self-employed.
2. Define the loss window (start/end) and partial capacity.
3. Compute gross loss and list assumptions.
4. List needed documents (pay stubs, W-2/1099, attendance).

## Output Template

```text
Wage Loss
- Rate: $[ ] per [hour/week/year]
- Dates missed: [ ] to [ ]
- Gross loss: $[ ]
- Assumptions: [ ]

Docs to request
- Pay stubs (pre/post)
- Attendance/time records
- Job description + restrictions
```
