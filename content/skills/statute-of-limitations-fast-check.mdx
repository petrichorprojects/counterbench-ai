## Steps

1. Identify claim type and jurisdiction.
2. List the default SOL window and common tolling flags.
3. Check whether any pre-suit notice deadlines apply (government defendants).
4. Produce the earliest plausible deadline (conservative).

## Copy/Paste Prompt

```text
Act as litigation intake support.

Task: Do a conservative statute-of-limitations + pre-suit notice screening.

Inputs:
- Jurisdiction: [STATE/COUNTRY]
- Venue: [COUNTY/CITY if relevant]
- Claim type: [NEGLIGENCE / MED MAL / PRODUCT / ETC]
- Incident date: [DATE]
- Plaintiff age/DOB: [DOB or ADULT]
- Any government entity involved? [YES/NO + WHO]

Output:
- Default SOL window(s) for this claim type
- Earliest likely filing deadline (conservative)
- Any notice-of-claim or admin exhaustion deadlines to confirm
- Tolling issues to check (minor, discovery rule, incapacity, fraud, etc.)
- 5 follow-up questions to confirm the deadline

If you are unsure, say so explicitly and list what authority you would check next.
```

## Quality Checks
- Includes a conservative deadline (earliest plausible).
- Separates SOL vs notice deadlines.
- Lists the exact missing facts to confirm.
