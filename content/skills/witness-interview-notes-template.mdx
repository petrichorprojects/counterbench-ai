## Steps

1. Capture identification + relationship + bias flags.
2. Build a timeline using anchors (time, location, sequence).
3. Record exact phrases for key admissions.
4. List follow-ups: other witnesses, documents, photos, video.

## Output Template

```text
Witness Interview

Who: [ ]
Relationship/Bias: [ ]

Timeline (anchors)
- …

Key facts (verbatim where possible)
- …

Credibility notes
- …

Follow-ups / evidence leads
- …
```
