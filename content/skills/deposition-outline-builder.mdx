## Steps

1. Define the 3 outcomes you need from this depo.
2. Group by: background, events, policies/practices, admissions, damages.
3. Attach exhibits to each topic.
4. Add impeachment hooks.

## Copy/Paste Prompt

```text
You are preparing a deposition outline.

Inputs:
- Witness: [NAME/ROLE]
- Case theory: [1-2 sentences]
- Defense themes: [PASTE]
- Key exhibits available: [LIST]

Output:
- Deposition outline with sections
- For each section: objectives, key questions, expected admissions, exhibits to use
- Add an "Impeachment" subsection with 3–6 hooks

Keep questions short and sequenced (foundation -> lock-in -> contradiction).
```

## Quality Checks
- Each section has a purpose (admission to obtain).
- Exhibits are pre-assigned to questions.
- Includes an impeachment plan.
