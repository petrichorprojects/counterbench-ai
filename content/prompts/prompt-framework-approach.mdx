## Prompt

```text
Use the APPROACH framework to create a communication plan and a draft message.

A — Audience:
{{audience}}

P — Purpose:
{{purpose}}

P — Priorities (top 3):
{{priorities}}

R — Results (what success looks like):
{{results}}

O — Openness (where you want feedback / what you’re uncertain about):
(Fill in if relevant.)

A — Assumptions:
{{assumptions}}

C — Concerns (risks, objections, sensitivities):
{{concerns}}

H — Help needed (what you want the recipient to do next):
(Fill in if relevant.)

Return:
1) A one-page plan (bullets) following APPROACH order
2) A draft message (email or memo, as appropriate)
3) 3-7 clarifying questions to improve accuracy
```
