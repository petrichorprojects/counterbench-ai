## Prompt

```text
Respond using the STAR framework. Use concrete details and avoid generic advice.

Topic:
{{topic}}

Scenario / background:
{{scenario}}

Constraints (time, jurisdiction, audience, tone, word count):
{{constraints}}

Output format (required):
## Situation
- ...
## Task
- ...
## Action
- ...
## Result
- ...

Rules:
- If inputs are missing, ask up to 5 clarifying questions first.
- If you make assumptions, label them explicitly.
```
