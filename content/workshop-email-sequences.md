**Subject Line A:** The one prompt that saves 20 minutes every time
**Subject Line B:** Try this before your next document review

**Preview text:** From Module 2 — but with a twist you can use today.

**Body:**

Hi {{first_name}},

Quick one. In Module 2 we built prompts using the Role + Context + Task + Format + Constraints framework. Here's a version I've refined since the workshop that consistently outperforms what we did in class:

```
You are an experienced litigation paralegal at a mid-size firm.
I'm going to paste a [document type]. Extract the following:
1. Key parties and their roles
2. Critical dates and deadlines
3. Obligations or commitments made
4. Anything that contradicts [specific document/statement]

Format: Numbered list under each header. Flag ambiguities separately at the end.
Constraints: Do not summarize — extract only. If something is unclear, say so.
```

The difference is that last line — "If something is unclear, say so." Most prompts let the AI fill gaps with confident-sounding guesses. This one forces it to flag uncertainty, which is exactly what you want when real case work is on the line.

Try it on something you're working on today. Takes 2 minutes to set up, saves 20 on the back end.

Hit reply if you run into something interesting — I read every response.
